## How can I edit this code?

There are several ways of editing this application.

---

### Use your preferred IDE (Local Development)

You can work locally using any IDE such as VS Code, WebStorm, or IntelliJ.

The only requirement is having **Node.js and npm** installed.

You can install Node.js using **nvm**:  
https://github.com/nvm-sh/nvm#installing-and-updating

Follow these steps:

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd recoverai-hub-main

# Step 3: Install the dependencies
npm install

# Step 4: Start the development server
npm run dev
